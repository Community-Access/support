# Contributing To Git Going Open Support

Thank you for helping learners and alumni.

## Scope

This repository is for support operations, not curriculum source development.

Allowed:

- Support issues
- Troubleshooting guidance
- Q&A discussion threads
- FAQ improvements

Not allowed:

- Curriculum content edits
- Template workflow code changes
- Security disclosures

## Triage Steps

1. Add an appropriate label.
2. Ask one clarifying question if details are missing.
3. Provide a concrete next action.
4. Mark resolved when user confirms outcome.

## Security And Privacy

- Never request tokens or credentials.
- Never ask users to post private repository links publicly.
- Route security issues to the security policy in the main workshop repository.
